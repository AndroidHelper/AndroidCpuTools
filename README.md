# 本工程基于Android 7.1 MTK平台，因隐藏类SDK获取不到，故需要在Android源码下编译才可以正常编译
本工程作用作为代码参考，目录有 CpuRunTools.apk，方便自行安装与看效果

目前Android各个系统版本在CPU的接口控制上变动性各有千秋，其他系统版本情况未经过测试哦
